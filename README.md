## Features

- Clean Design
- Responsive
- Featured Rooms
- Rooms list
- Room filteration
- Single page of Room
- Room booking with Specific Account
- Private Routes
- Authentication
- Validation
- Booking details page
- Booking tracking





## Tech Stack

**Client:** React, Redux, React-Bootstrap , Bootstrap

**Server:** Firebase Realtime Database, Firebase Authentication


## Run Locally

Clone the project

```bash
  git clone https://github.com/Nabia-Sheikh/hotel-management-system.git
```

Go to the project directory

```bash
  cd hotel-management-system
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
